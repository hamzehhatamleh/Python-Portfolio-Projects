# Python Portfolio: Days 20-80 (Angela Yu)

This repository tracks my intermediate to advanced practical software development progress through the "100 Days of Code" Python course. 

I have intentionally excluded the absolute beginner fundamentals (Days 1–19) to showcase intermediate programming concepts, automation, and real-world projects.

## 🛠️ What's Inside:
* **Days 20-23:** OOP & Game Development (Snake, Pong, Turtle Crossing)
* **Days 24-31:** Local File Handling & GUI Building (Tkinter, Password Managers)
* **Days 32-60:** Working with APIs, Data Scraping, and Web Development Basics
* **Days 61-80:** Advanced Python applications and Data Science introductions
